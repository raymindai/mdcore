# 30s demo video — script + shot list

**Target length**: 30 seconds. Hard cap at 35.
**Voice over**: founder's voice, scripted but read like a friend explaining over coffee. No music underneath OR a *very* quiet pad — no swells.
**Shot rhythm**: 5 cuts of ~6 sec each. The screen does the talking; the VO names what we're seeing.
**Aspect**: 16:9 master + 9:16 vertical cut (Twitter / TikTok)
**Style**: dark theme (matches the app), no zoom-bounces, no kinetic typography. Real screen capture only.

---

## Storyboard

### Shot 1 (0–6s) — The problem in one frame

**On screen**: split screen. Left half: ChatGPT giving a great answer. Right half: a Google Drive / Notion folder full of orphan files with bad names. Both fading to "you'll never find this again."

**VO**:
> Every day you ask AIs hard questions and get back useful answers. And every day you close the tab.

---

### Shot 2 (6–12s) — Capture

**On screen**: Chrome ext on a Claude chat → click mdfy button → URL pops in a toast → cuts to mdfy.app showing the doc rendered cleanly.

**VO**:
> mdfy captures it. One click, permanent URL.

---

### Shot 3 (12–18s) — Bundle / Hub

**On screen**: sidebar showing 8-10 docs cluster into a Bundle. Bundle viewer briefly. Quick cut to `mdfy.app/hub/<you>` page with the stats strip (40 docs · 40 concepts · ~60k tokens).

**VO**:
> Captures group into a hub. Concept index links them. Your whole memory at one address.

---

### Shot 4 (18–25s) — Deploy across AIs

**On screen**: copy URL gesture. Three windows in quick succession: paste into Claude, paste into Cursor `.cursorrules`, paste into a ChatGPT custom GPT instructions box. Each one immediately responds with a citation back to a specific doc.

**VO**:
> Paste the URL into Claude. Cursor. ChatGPT. Same URL. Same context. Every AI reads it the same way.

---

### Shot 5 (25–30s) — Close

**On screen**: just the logo and the tagline center-screen. `mdfy.app` underneath.

**VO**:
> Your AI memory, deployable to any AI. Try it — no signup. mdfy.app.

---

## Production notes

- Record at 1440p so we can crop to 1080p without quality loss
- Cursor screen-recording shows the system clock / battery — mask it
- For the "paste into Cursor" shot, use a real project with a `.cursorrules` already containing `# Context\nhttps://mdfy.app/hub/...` — looks more honest than typing it live
- Don't fake the AI's response time. If Claude takes 3 seconds, let it take 3 seconds — speeds up later in edit if needed
- The Hub page screenshot should show real stats, not "Lorem ipsum 999 docs"
- End frame holds for 1.5 seconds with the URL legible — this is the call to action, don't cut it short

## Distribution channels

- mdfy.app home page: embed at the top as a muted-by-default autoplay
- About page: embed in the hero
- Twitter: 16:9 with subtitles burned in (autoplay is muted on TL)
- LinkedIn: 16:9 with subtitles
- TikTok / Instagram Reels: 9:16 crop
- HN: just a static thumbnail in the Show HN post — link to the video on the home page

## A note on AI-narrated voice

Don't. People hear AI voice in seconds and tune out. If founder voice is the issue, get a friend with a clean mic. Authenticity beats polish for this audience.
